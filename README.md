# RingCentral JS SDK in Shared Worker

Run RingCentral JS SDK in Shared Worker

## Test in local

Clone this project:

```
$ git clone git@github.com:embbnux/ringcentral-shared-worker.git
```

Install Dependencies:

```
$ yarn
```

Build JS files:

```
$ yarn build
```

Start demo server:

```
$ yarn start
```

Test at `http://localhost:8080/demo/`.
